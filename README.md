# 医院管理系统 java563

## 项目概述

医院管理系统是一个基于Spring Boot框架的前后端分离的系统。该系统致力于为医院提供全面的管理解决方案，涵盖医生、患者以及行政管理等多个方面。

## 技术栈

- **前端：** Vue + Element UI
- **后端：** Java + Spring Boot

## 功能模块

系统包括三大角色：管理员、医生和病人，以下为各角色功能模块介绍。

### 管理员

- **首页：** 显示系统概览。
- **医生管理：** 对医生信息进行管理。
- **患者管理：** 对患者信息进行管理。
- **挂号管理：** 处理挂号相关事务。
- **药物信息：** 管理药物相关信息。
- **检查项目：** 管理各类检查项目。
- **病床信息管理：** 管理病床分配与状态。
- **排班信息管理：** 对医生排班进行设置。
- **数据统计功能：** 统计各类医疗数据。

### 医生

- **首页：** 显示个人工作概览。
- **今日挂号列表：** 查看当日挂号患者。
- **历史挂号列表：** 查看历史挂号记录。
- **住院申请管理：** 管理住院申请。
- **个人信息查询：** 查看与编辑个人资料。

### 病人

- **首页：** 显示患者相关信息。
- **预约挂号：** 提供在线挂号服务。
- **我的挂号：** 查看挂号记录。
- **住院信息：** 查看与管理住院信息。
- **个人信息功能：** 管理个人资料。

## 系统角色
